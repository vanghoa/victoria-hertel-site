# Here to set up the local development first time on your computer

1. Install Visual Studio Code
    - Install Prettier - Code formatter extension
    - Install MDX extension
2. Install Brew
    - https://brew.sh/
3. Install Node.js
    - https://nodejs.org/en/download/package-manager
4. Install Git
    - https://git-scm.com/download/mac brew install git
    - Set up Git credentials
5. Clone the Git repository folder
6. Install all the npm dependencies
    - Open the cloned folder in Visual Studio Code,
    - Open the terminal with [shortcut Ctrl + ` (backtick)] or [Terminal > New Terminal]
    - Type 'npm install', press 'Return' and wait
